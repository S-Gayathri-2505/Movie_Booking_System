# 🎬 Console-Based Movie Ticket Booking System

A **Console-Based Movie Ticket Booking System** developed using **Core Java** that simulates the movie ticket reservation process through a menu-driven interface. The application provides separate functionalities for administrators and users, enabling efficient management of movies, seat reservations, bookings, cancellations, and booking history.

---

## 📌 Project Overview

The purpose of this project is to automate the movie ticket booking process and eliminate the limitations of manual booking systems. The application demonstrates the practical implementation of **Core Java**, **Object-Oriented Programming (OOP)**, and **Java Collections Framework** concepts in a real-world scenario.

This project was developed as part of Java programming training to strengthen problem-solving and software development skills.

---

## ✨ Features

### 👨‍💼 Administrator Module
- Admin Login

### 👤 User Module

## ⚙️ How to Run

### 1️⃣ Clone the Repository

```bash
git clone https://github.com/S-Gayathri-2505/Movie_Booking_System.git
```

### 2️⃣ Open the Project

Open the project in **Visual Studio Code** or any Java IDE.

